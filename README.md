## JupyterNotebookを起動する

事前に[Dockerのインストール](https://docs.docker.com/install/)が必要です。

### Clone Repository

```
$ git clone https://github.com/masayuki14/hannari-python-3.git
```

### Build image and run docker

```
$ sh run.sh
```

### Access localhost

[http://localhost/](http://localhost/)
